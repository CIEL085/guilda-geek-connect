💖 Projeto Rede Social & App de Namoro

Bem-vindo ao repositório do meu projeto de rede social integrada a um aplicativo de namoro.
O objetivo é criar uma plataforma moderna, interativa e responsiva, que permita usuários se conectarem, interagirem e encontrarem potenciais parceiros.

📌 Sobre o Projeto

Este projeto combina recursos de uma rede social tradicional com funcionalidades de um app de relacionamento, oferecendo:

👤 Criação de perfis personalizados

💬 Sistema de mensagens em tempo real

❤️ Match baseado em interesses e preferências

📸 Upload e gerenciamento de fotos

🔔 Notificações inteligentes

🌐 Interface responsiva para web e mobile

🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

Vite – Build rápido e otimizado

React – Biblioteca para construção da interface

TypeScript – Tipagem estática para maior segurança no código

Tailwind CSS – Estilização ágil e moderna

shadcn/ui – Componentes prontos e acessíveis para React
